SSGen - Static Site Generator

This is a single-script site generator that uses a set of source files, templates and a config file to generate the site. No alternate markup languages are used, no Javascript is involved (unless the user desires it) and the script does not have any external dependencies aside from Perl 5.

term - VT100 Compatible Terminal Tools

This is a very small C library for terminal color formatting, cursor positioning, input mode manipulation etc. It can likely be used with most of the modern terminals.
